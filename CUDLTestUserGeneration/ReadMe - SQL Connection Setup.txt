When using Windows Authentication Integrated Security

1. Download the sqljdbc_9.2.1.0_enu.zip from Microsoft Site
2. Extract zip file
3. copy sqljdbc_4.0/enu/auth/x64/sqljdbc_auth.dll to Java bin and lib folders

Example, find java instaltion folder and move file to the following locations:
C:\Program Files\Java\jre1.8.0_281\bin
C:\Program Files\Java\jre1.8.0_281\lib
